# AI 보안 솔루션 경쟁사 분석표

## 1. 보안박람회 7개 업체 확인 결과 요약
* **대상 업체**: 안랩, 안랩클라우드메이트, 샌즈랩, 모니터랩, 지란지교, 이글루, 윈스
* **현황 특징**: 
  - 대부분 네트워크·웹 보안의 연장선에 머물러 있음.
  - 개발자 환경(API/IDE)과 LLM 내부 동작까지 통제하는 **‘개발자용 AI 보안’은 아직 부재**함.
  - 공통적으로 토큰/사용량 제어 등의 **비용을 관리하거나 가시화하는 기능(FinOps)은 전무**한 상황.
* **가장 돋보이는 사례 (안랩 - 시큐어브릿지)**:
  - 현재 시중에서 가장 잘 구현된 형태로 보임 (단, 상용 상품은 아니며 고객사 요청 시 제공. 향후 오픈소스로 공개 예정).
  - IDE 확장(플러그인)과 API 서버를 통해 개발자의 LLM 호출을 중간에서 통제하는 구조.
  - LangChain 기반으로 맥락 필터링(sLLM) 구현 (사용 모델은 기밀).
