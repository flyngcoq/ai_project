# 🎨 UHD PPT Design Guide (v6.0)
**Objective:** Deliver high-impact strategic insights through top-down logic and precision data visualization.

---

## 🏗 1. Strategic Storyline & Logic

### 1.1 The Pyramid Principle (Top-Down)
- **Leading with the Answer:** 결론(Primary Insight)을 가장 먼저 제시하고, 그 아래에 근거를 배치한다.
- **MECE Structuring:** 모든 근거는 '중복 없이, 누락 없이(Mutually Exclusive, Collectively Exhaustive)' 구조화되어야 한다.

### 1.2 SCQA Narrative Model
- **Situation (상황):** 현재의 맥락 제시.
- **Complication (전개):** 발생한 문제나 변화.
- **Question (질문):** 해결해야 할 핵심 과제.
- **Answer (대답):** 우리의 전략적 제안.

## 📊 2. Strategic Data Visualization

### 2.1 Chart Selection Matrix
| 목표 | 추천 차트 | 활용 예시 |
| :--- | :--- | :--- |
| **변화 원인 분석** | **Waterfall (폭포)** | 수익 증감 요인, 예산 변동 브릿지 |
| **2차원 비중 분석** | **Mekko (메코)** | 시장 점유율, 고객 세그먼트 분석 |
| **비교 및 랭킹** | **Horizontal Bar** | 항목별 성과 비교 |
| **프로세스/시간** | **Gantt / Chevron** | 실행 로드맵, 타임라인 |

### 2.2 Color Semantics (의미론적 색상 활용)
- **Positive/Growth:** Emerald Green / Blue
- **Negative/Decline:** Red / Orange
- **Total/Standard:** Deep Gray / Navy

## 📐 4. Content Density & Vacancy Rule (v7.0)
- **Maximum 20% Vacancy:** 모든 슬라이드의 시각적 공백율은 20% 미만이어야 한다. 정보 부족으로 장표가 비어 보이는 것은 전문가로서의 수치다.
- **No Placeholder Slides:** "슬라이드 X: 제목"만 있는 껍데기 장표는 즉시 반려한다. 반드시 표, 차트, 통합 도형으로 내용을 빽빽하게 채워야 한다.
- **Micro-Data Mapping:** 한 슬라이드 내에 최소 7~10개의 유니크 데이터 포인트(가격, 시간, 규칙 등)를 배치하여 정보 밀도를 극대화한다.
- **Visual Satiety:** 표와 텍스트 상자를 병렬 배치하거나 2단 레이아웃을 활용하여 시각적 충만함을 확보한다.
